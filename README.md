  # 1️⃣ 프로젝트 소개
### 프로젝트 명 
**NovelKub**
### 프로젝트 소개  
Novel + Rumikub = NovelKub<br/>
저희의 프로젝트 게임 장르는 추리 퍼즐 게임입니다.<br/>
플레이어는 탐정이 되어 한 살인사건을 해결하기 위해 주변NPC를 통해<br/> 
단서를 찾아내서 사건의 마무리에 다가가는 과정을 그립니다.
### 게임유형 
추리 노벨 스토리 게임
### 개발기간 
2023.10.13 ~ 2023.10.20
